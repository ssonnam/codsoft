Title : Record Room & Report Generator
Description: The application is to provide a comprehensive and user-friendly platform that empowers educators to efficiently manage various aspects of department and student data and also to maintain real time updates Which is based on AQAR (Annual Quality Assurance Report)
Objective: The objective of this web application is to implement a form submission feature that captures user input and dynamically generates an Excel spreadsheet containing the submitted details. This functionality will enable seamless data collection and export for efficient record-keeping and analysis purposes.
Technology: HTML,CSS,JAVASCRIPT
